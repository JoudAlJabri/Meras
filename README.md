# Meras 🌱

Meras is a web-based platform designed to support users in exploring career paths, completing interactive challenges, and connecting with mentors.
The system is built with a **multi-role structure**, including **Explorer, Mentor, Guide, and Admin**, where each role provides a different experience within the platform.

---

## 🚀 Features

### 👩‍💻 Explorer

* 📊 Dashboard to track progress
* 🧩 Browse and complete challenges
* 🧭 Compass Quiz to discover career paths
* 👩‍🏫 View and connect with mentors
* 👤 Manage personal profile

### 👨‍🏫 Mentor

* View assigned explorers
* Provide guidance and support
* Interact with users through the platform

### 🧭 Guide

* Assist explorers in navigating their journey
* Monitor user progress and activities

### 🛠️ Admin

* Manage system users
* Oversee platform functionality and data

---

## 🛠️ Tech Stack

* **Frontend:** React + Vite
* **Routing:** React Router
* **Styling:** CSS + Bootstrap + React-Bootstrap
* **Icons:** React Icons
* **Charts/Data Visualization:** Recharts
* **Other Libraries:** Axios

---

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/JoudAlJabri/Meras.git
```

2. Navigate to the project folder:

```bash
cd Meras
```

3. Install dependencies:

```bash
npm install
```

---

## ▶️ Running the Project

Start the development server:

```bash
npm run dev
```

Then open your browser and go to:

```
http://localhost:5173
```

## 🧑‍💻 Usage

* Navigate through the application based on your role
* Explorer users can interact with challenges, quizzes, and mentors
* Mentors and Guides provide support and guidance
* Admin manages users and platform structure
* All navigation is handled through role-based layouts

## ⚠️ Notes

* Data is currently static or simulated
* The project runs locally using Vite
* Make sure Node.js is installed before running
